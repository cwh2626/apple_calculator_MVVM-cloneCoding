# 1.애플 계산기 클론코딩 - MVVM 패턴 적용
애플의 기본 애플리케이션인 '계산기'의 디자인과 동작을 참고하여 구현한 소스코드입니다. 100% 동일하지는 않지만, 대략적으로 비슷한 기능을 구현하였으며, MVVM 패턴에 맞추어 코드를 재구성하였습니다. 코드 재구성 시 사용된 기술로는 RxSwift, Combine, KVO 등이 있으며, 이들 라이브러리의 모든 기능을 사용하기보다는 View와 ViewModel을 연결해주는 바인딩 기능에 중점을 두었습니다. 각 라이브러리별로 브랜치를 나누어 작성하였습니다.

## 1.2 브랜치 구성
* [1] Basic - MVVM 패턴 적용 전
* [2] MVVM_KVO 
* [3] MVVM_RxSwift
* [4] MVVM_closure
* [5] MVVM_combine

## 1.3 사용법
'1.2 브랜치'를 참고하여 원하는 브랜치를 선택하고 로컬에 클론하면, '2. 구동영상'처럼 바로 사용할 수 있습니다.
